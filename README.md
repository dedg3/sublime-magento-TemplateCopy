TemplateCopy
=============

Sublime Text 2 Plugin to copy Magento Template files complete with folder path. 


### Settings

 (.sublime-project) File

    "settings":
    {
        "tc_theme": "mytheme",
        "tc_enterprise": "true"
    }


### Usage
-----

* Sidebar Right Click on Filename
* Right Click -> Context menu
* Right CLick on Open file tab

### Keybindings

	Mac OS X: CTRL+CMD+T 
	Windows:  CTRL+ALT+T
	Linux:    CTRL+ALT+T
